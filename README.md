# Vue combobox

Preview da aplicação:
https://contesini.github.io/vue-combobox/

Baixa as dependencias:
```bash
yarn ou npm i
```

Para rodar tests integração suba o servidor:
```bash
yarn serve ou npm run serve
```
depois execute:
```bash
npx cypress run
```

Para executar os tests unitarios execute:
```bash
yarn test:unit
```
